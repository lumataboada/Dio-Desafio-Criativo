# Dio-Desafio-Criativo
Extraindo Insights do Feedback de Clientes Bancários

Atue como Consultor de negócios 

Sua tarefa é analisar os feedbacks da pesquisa de satisfação sobre os produtos/serviços e chat de atendimento para identificar a necessidades de novos serviços/produtos

Contexto: O resultado será usado pela Diretoria de produtos para identificar a necessidade de criação de novos produtos bancários

Dados disponíveis: A base contém dados de pesquisa de satisfação sobre cada serviço/produto ou chat de atendimento do banco.
	Notas 1 até 5
	Data pesquisa
	Flag de que se expectativa foi atendida.
	comentário em texto.

Instruções de análise:

Classifique os feedbacks por serviços/produto e chat de atendimento, e se expectativa foi atendida
Identifique os principais padrões, problemas, elogios e oportunidades.
Aponte evidências nos dados fornecidos.
Sugira ações práticas para a Diretoria de produtos 
Formato da resposta: 
	Uma tabela com os principais serviços bancários avaliados melhor nota na pesquisa de satisfação,
	uma tabela com os principais serviços bancários avaliados pior nota na pesquisa de satisfação,
	Uma tabela com as principais recomendações e possíveis necessidades de novos produtos e

	Uma tabelas de 3 até 5 avaliações de clientes para cada tabela acima.

	Um breve resumo da pesquisa de satisfação

Restrições:

Use apenas os dados fornecidos.
Não invente números, causas ou conclusões.
Não exponha dados pessoais ou sensíveis.
Informe limitações quando os dados não forem suficientes.
Use linguagem executiva voltada para tomada de decisões
